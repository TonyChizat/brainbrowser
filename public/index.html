BrainBrowser is an online visualization tool for brain imaging. It allows researchers to explore datasets of various types such as the precalculated MACACC(Mapping anatomical correlations across cerebral cortex) or data provided by the user in obj, text, minc, nifty format using the surface viewer.

Brainbrowser is integrated into the CBRAIN platform for a completely web based research workflow. The combination of CBRAIN and BrainBrowser eliminates the pain of having to wrestle with command line processing tools, transfering datasets manually to High Performance computing centers and local storage of results to use with difficult to install desktop visualization tools. All this within one interface. For more information on CBRAIN visit the CBRAIN website
