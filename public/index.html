Brainbrowser is a central visualization component of the CBRAIN platform. CBRAIN is the CANARIE-funded, integrated computational platform for brain imaging research. It combines the networking capabilities of the CANet4 network with the processing power of High Performance Computing centres across Canada and Europe and cutting-edge visualization tools such as BrainBrowser, and makes them available through an intuitive, graphical, web browser-based interface.

This demonstration of BrainBrowser can be used in one of the following ways:

  1. Surface Viewer: Load and manipulate 3D brain surface or DTI models.
  2. Macacc Viewer: The Macacc dataset is 1TB set of correlation maps for cortical thickness, cortical area and cortical volume.

Go to the Viewers tag to start using BrainBrowser.

BrainBrowser is web-based, 3D visualization tool for neuroimaging. Using web-standard technologies, such as WebGL and HTML5, it allows for real time manipulation and analysis of 3D neuroimaging data whether it be precalculated maps, such as the MACACC data set (Mapping Anatomical Correlations Across Cerebral Cortex), or models provided by the user in MNI object format and data in one of the many currently supported formats (Minc, Nifty, object files, plain text).
