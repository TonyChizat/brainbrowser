Configuration

The Volume Viewer is configured by defining the object BrainBrowser.config.volume_viewer. Currently the only properties available for configuration are the color scales which are configured to define their name, the URL at which the color scale file is located, and, optionally, the color to use for the cursor when the defined color scale is active:

BrainBrowser.config = {

  volume_viewer: {
    color_scales: [
      {
        name: "Spectral",
        url: "/color_scales/spectral.txt",
        cursor_color: "#FFFFFF"
      },
      {
        name: "Gray",
        url: "/color_scales/gray_scale.txt",
        cursor_color: "#FF0000"
      }
    ]
  }

}
