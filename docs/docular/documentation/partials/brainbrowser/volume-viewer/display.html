display

Description

Object representing an individual canvas display.

Properties

  • volume

    The volume being displayed.

  • axis

    The name of the axis being displayed (xspace, yspace or zspace).

  • slice

    The slice currently being displayed.

  • canvas

    Reference to the canvas area used for drawing.

  • context

    The 2D context of the canvas.

  • image_center

    The x and y coordinates of the center of the slice currently being displayed.

  • zoom

    The current zoom level of the display.

  • cursor

    The current x and y coordinates of the cursor.

  • mouse

    The current x and y coordinates of the mouse.
