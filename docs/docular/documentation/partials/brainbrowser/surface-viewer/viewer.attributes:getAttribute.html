getAttribute ( attributes in module viewer )

Description

Retrieve the value of an attribute.

The viewer object can maintain an arbitrary set of key-value to aid in the functioning of various parts of the system.

Currently, the following attributes are used by the Surface Viewer:

  • clamp_colors: clamp intensity values outside the current range to maximum and minimum values of the color map.
  • flip_colors: invert the intensity to color map relationship.
  • fixcolorrange: maintain the current intensity range, even if new data is loaded.

Parameters

  • name – {string} –

    Name of the attribute to retrieve.
