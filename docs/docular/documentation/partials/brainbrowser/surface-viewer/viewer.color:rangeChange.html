rangeChange ( color in module viewer )

Description

Update the range of colors being applied to the current model.

Parameters

  • min – {number} –

    Minimum value of the range.

  • max – {number} –

    Maximum value of the range.

  • clamped – {boolean} –

    Should values be clampled to the min/max range?

  • options – {object} –

    Options for the range change, which include the following:

    • complete Callback function to call when the color update is done.
