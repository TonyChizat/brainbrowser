pick ( rendering in module viewer )

Description

Given an x and y coordinate on the viewer canvas, returns information about the the point of intersection with a displayed object.

Parameters

  • x – {number} –

    The x coordinate on the canvas.

  • y – {number} –

    The y coordinate on the canvas.

Returns

{object} –

If an intersection is detected, returns an object with the following information:

  • object The THREE.Object3D object with which the the click intersected.
  • point A THREE.Vector3 object representing the point in 3D space at which the intersection occured.
  • index The index of the intersection point in the list of vertices.

Otherwise returns null.
