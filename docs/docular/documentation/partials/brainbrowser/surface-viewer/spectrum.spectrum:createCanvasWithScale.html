createCanvasWithScale ( spectrum in module spectrum )

Description

Create a canvas spectrum from the given colors and provide the scale for it.

Parameters

  • min – {number} –

    Min value of the color data.

  • max – {number} –

    Max value of the color data.

  • colors – {array} –

    Array of color data.

  • flip – {boolean} –

    Array of color data.
