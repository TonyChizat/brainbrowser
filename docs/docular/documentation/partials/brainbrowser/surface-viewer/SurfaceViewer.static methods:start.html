start ( static methods in module SurfaceViewer )

Description

The start() function is the main point of entry to the Surface Viewer. It creates a viewer object that is then passed to the callback function supplied by the user.

  BrainBrowser.SurfaceViewer.start("brainbrowser", function(viewer) {
    
    //Add an event listener.
    viewer.addEventListener("displaymodel", function(model) {
      console.log("We have a model!");
    });

    // Start rendering the scene.
    viewer.render();

    // Load a model into the scene.
    viewer.loadModelFromUrl("/models/brain_surface.obj");
     
    // Hook viewer behaviour into UI.
    $('#wireframe').change(function(e) {
      viewer.setWireframe($(this).is(":checked"));
    });

  });

Parameters

  • element_id – {string} –

    ID of the DOM element in which the viewer will be inserted.

  • callback – {function} –

    Callback function to which the viewer object will be passed after creation.
