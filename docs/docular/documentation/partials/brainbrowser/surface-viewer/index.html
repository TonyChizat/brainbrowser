The BrainBrowser Surface Viewer is a tool for displayed and manipulating 3D datasets in real time. Basic usage consists of calling the start() method of the SurfaceViewer module, which takes a callback function as its second argument, and then using the viewer object passed to that callback function to control how models are displayed: 

   BrainBrowser.SurfaceViewer.start("brainbrowser", function(viewer) {
     
     //Add an event listener.
     viewer.addEventListener("displaymodel", function(model) {
       console.log("We have a model!");
     });

     // Start rendering the scene.
     viewer.render();

     // Load a model into the scene.
     viewer.loadModelFromURL("/models/brain_surface.obj");
      
     // Hook viewer behaviour into UI.
     $('#wireframe').change(function(e) {
       viewer.setWireframe($(this).is(":checked"));
     });

   });
