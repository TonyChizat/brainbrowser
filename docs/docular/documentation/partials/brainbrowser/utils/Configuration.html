Configuration

BrainBrowser configuration is done by creating an object, BrainBrowser.config, with two object properties, surface_viewer and volume_viewer that define the configuration options for each of the tools. For more information on particular options for each tool, see the Surface Viewer and Volume Viewer configuration sections.

BrainBrowser.config = {

  surface_viewer: {
    // ...
  }, 

  volume_viewer: {
    // ...
  }
  
}
