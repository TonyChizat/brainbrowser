checkConfig ( utils in module BrainBrowser )

Description

Check for the existance of a configuration parameter. Argument is a dot-separated list of nested configuation parameter keys. Returns whether given parameter is set in the current configuration. For example: 

  BrainBrowser.utils.checkConfig("surface_viewer.worker_dir");
  // returns true if BrainBrowser.config.surface_viewer.worker_dir has been set.

Parameters

  • config_string – {string} –

    Dot-seperated list of nested configuration options to check for.

Returns

{boolean} –

Whether or not the configuration option is defined.
