createCanvasWithScale ( color_map in module color_map )

Description

Create a canvas color_map from the given colors and provide the scale for it.

Parameters

    Min value of the color data.

  • max – {number} –

    Max value of the color data.

  • colors – {array} –

    Array of color data.

  • flip – {boolean} –

    Array of color data.
